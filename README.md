# Tymeless ADHD ARMY T-Shirt Site

Static storefront prototype for Tymeless ADHD ARMY black and pink T-shirts.

## Development

Install dependencies:

```powershell
npm install
```

Start the dev server:

```powershell
npm run dev
```

Build the deployable draft:

```powershell
npm run build
```

The build output is written to `dist/`.

Tee and hero mockups are generated locally into `assets/` by:

```powershell
powershell -ExecutionPolicy Bypass -File tools/generate-assets.ps1
```
